pennair2.utm.to_latlon(easting, northing, zone_number, zone_letter=None, northern=None, strict=True)¶

This function convert an UTM coordinate into Latitude and Longitude

easting: int
Easting value of UTM coordinate
northing: int
Northing value of UTM coordinate
zone number: int
Zone Number is represented with global map numbers of an UTM Zone Numbers Map. More information see utmzones [1]_
zone_letter: str
Zone Letter can be represented as string values. Where UTM Zone Designators can be accessed in [1]_
northern: bool
You can set True or False to set this parameter. Default is None
pennair2.utm.from_latlon(latitude, longitude, force_zone_number=None)¶

This function convert Latitude and Longitude to UTM coordinate

latitude: float
Latitude between 80 deg S and 84 deg N, e.g. (-80.0 to 84.0)
longitude: float
Longitude between 180 deg W and 180 deg E, e.g. (-180.0 to 180.0).
force_zone number: int
Zone Number is represented with global map numbers of an UTM Zone Numbers Map. You may force conversion including one UTM Zone Number. More information see utmzones [1]_
